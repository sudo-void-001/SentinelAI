# 🛡️ SentinelAI

> **AI-Powered Cyber Threat Intelligence Platform**

SentinelAI is a cybersecurity platform that automates the collection, analysis, and delivery of cyber threat intelligence.

It aggregates cybersecurity news and vulnerability data, leverages AI to generate concise threat summaries, and provides security professionals with actionable insights through dashboards, scheduled reports, and automated notifications.

---

## ✨ Key Features

- 📰 Multi-source cybersecurity news aggregation
- 🚨 CVE monitoring and vulnerability tracking
- 🤖 AI-powered threat summarization
- 📧 Automated daily email threat digests
- 👥 Multi-user platform with role-based access
- 🔐 JWT Authentication & Authorization
- ⚙️ Scheduled background automation
- 📊 Interactive dashboard
- 🛡️ Admin management system
- 📈 Production-style backend architecture
